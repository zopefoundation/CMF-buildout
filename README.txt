============
CMF.buildout
============
-------------------------
Build CMF 2.3 + Zope 2.13
-------------------------

Introduction
============

Builds CMF 2.3 with Zope 2.13 from develop eggs located in ``src``.

Dependencies
============

Requires Python 2.6

Usage
=====
::

  $ python2.6 bootstrap/bootstrap.py
  $ ./bin/buildout
  $ ./bin/test
  $ ./bin/instance
